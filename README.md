kits
-----
kits 是一个公共库，包含了常见的框架组件，易用工具

安装
-----
```
git clone git@gitlab.tools.lwork.com:golang/kits.git ${GOPATH}/src/lwork.com/kits
```
目录结构
-----
```
${GOPATH}
├── bin
├── pkg
└── src
    ├── github.com
    ├── gopkg.in
    └── lwork.com
        ├── bw
        │   ├── manager
        │   ├── rcrc
        │   └── rcre
        ├── fw
        │   ├── hds
        │   └── tss
        ├── gw
        │   ├── ftss
        │   └── otss
        ├── kits
        └── tw
```
